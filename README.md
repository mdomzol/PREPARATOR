# 🛠️ Preparator

**Preparator** is a modern Windows desktop tool built with WPF and C# designed to quickly prepare and configure a workstation.

It provides a clean UI and modular approach to system setup, including power plans, software installation, and system optimization.

---

## ✨ Features

- ⚡ One-click system preparation (planned)
- 🖥️ System configuration tools
- 🔋 Power plan management
- 📦 Software installation management
- 🎨 Modern UI (WPF + MVVM)
- 🧭 Sidebar navigation with active state
- ✨ Smooth view transitions

---

## 🧱 Architecture

The application is built using the **MVVM (Model-View-ViewModel)** pattern.
